- ポークカレー作り方
  - まず、豚肉を適当な大きさに切る。
　　 - じゃがいもと人参を皮をむいて、食べやすい大きさに切る。
  - 玉ねぎもみじん切りにする。
  - 鍋に油を熱し、豚肉を炒める。
  - 豚肉に火が通ったら、玉ねぎを加えて炒める。
  - 野菜がしんなりするまで炒めたら、水またはだしを加える。
  - 野菜に火が通るまで煮る。
  - じゃがいもと人参を加え、柔らかくなるまで煮る。
  - カレールーを加えて溶かし、とろみがつくまで煮る。
  - 味を見て、塩とこしょうで調整する。
  - 器に盛り付けて完成！
